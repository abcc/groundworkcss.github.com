Groundwork Components

The Grid

The Groundwork grid system is the most advanced responsive grid system in the world. Flexible fluid column sizing based on simple fractions that can be infinitely nested with elegant responsive adaptation. What more could you want?

Media Queries

The Groundwork grid system has two thresholds that trigger adaptations for responsive layouts. One for small tablet-like devices less than 768 pixels wide and one for mobile phones and other handheld devices with horizontal resolutions 480 pixels and below.

Javascript Components

ResponsiveText

An easy to use tool to set font sizes responsively based on their container width. Use responsiveText.js to have scalable headlines, build responsive tables and more!

Popovers

Responsive popover bubbles with smart positioning, resizing with window resize and smart repositioning on window scroll.

Tooltips

Responsive tooltips with smart positioning, resizing with window resize and smart repositioning on window scroll.

Orbit

A powerful image and/or content slider created by Zurb that works responsively and includes tons of useful options like: timers, paddles, bullet thumbs and more.

ColorBox

A lightweight customizable lightbox plugin.
